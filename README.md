# Portatif-sogutucu-cihaz-
Araç içi veya piknikte içeceklerinizi serin tutmak için tasarlanmış bir cihazdır
Termoelektrik soğutucular tarafından içerisi soğuk tutulurken dışarısnın ısısı fanlarla yayılmaktadır
Bunun sonucunda içerisini daima serin tutmayı başardık. 12V 10A lik bir kaynaktan beslenen cihazımızda
30 dk süre sonunda 13 C dereceye ulaşılmıştır. Sistemin verimliliğinin arttırılması için geliştirmelere açık olan projede
bilgisayar fanı soğutucuları ile hava soğutması sağlanmıştır. Sıvı soğutma ile daha iyi sonuçlar elde edilebilir.
